How to install

1)Install apache-tomcat 
2)Put the converter folder to \apache-tomcat-<version>\webapps.
3)Run the tomcat.
4)project on -> http://localhost:8080/ShoppingCart/

In order to make changes go to converter/WEB-INF/src and change tha java files.The put the .class files in converter/WEB-INF/classes/ex
